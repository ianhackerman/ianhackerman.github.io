ATM

Mar. 25, 2019

The ATM Python Project is a program that emulates the functions of a atm with the programming language Python. One difficulty I had while making this program was a bug where typing letters instead of numbers while depositing or withdrawing crashed the program. To fix this I used the Try and Except functions of python. If an input crashes the program on try, the program continues by executing except. Another difficulty was creating multiple accounts for the atm. I created fixed accounts to login to that each had their own balances.
