Arduino Piano

May. 30, 2019

Initially, Vance and i wanted to create a board for a Simon Says type game but I later decided to make a piano. As you press a button, the corresponding led lights up and a tone plays. We thought that the buzzer was annoying so Vance had the idea to add a volume knob. At first, we had a problem where the LEDs would detect tables instead of lighting up after a button it pressed, I fixed this by adding INPUT_PULLUP to the button setup. This project was a collaborative effort between Vance and I.
