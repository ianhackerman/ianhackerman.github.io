SkateBro

Oct. 4, 2018

The graphics of this app feature simple characters drawn in MS Paint. I used an older version of Thunkable as it has more features, such as moving sprites around freely. A difficulty I encountered was the bad guy not speeding up at 3 second intervals, I fixed this by adding a block to set his speed to the clock block. Another difficulty I had while making this app was that the start screen did not start correctly, I decided that the app should start in the game instead.
